## 🖱️ How to Use It

After setup, use a 3-finger swipe to switch spaces.

- Swipe left to move to the next space
- Swipe right to move to the previous space
- Use it on the trackpad
- Keep using your normal macOS space layout

The app works in the background. You do not need to open it each time you want to switch spaces.

## 💻 What You Need

iss is for macOS.

A good setup includes:

- macOS with Space support turned on
- A trackpad or Magic Trackpad
- Three-finger gestures enabled
- Enough permission for the app to read touch input
- A recent version of macOS for best results

If you use a MacBook, the built-in trackpad should work well. If you use a desktop Mac, a trackpad gives the best experience.

## ⚙️ Before You Start

Make sure Spaces are turned on in macOS:

1. Open System Settings.
2. Go to Desktop & Dock.
3. Find Mission Control settings.
4. Turn on spaces-related gesture options if they are available.
5. Check that your trackpad gesture settings are not blocking three-finger swipes.

If another app uses the same gesture, you may need to change that app first.

## 🧩 Troubleshooting

If the swipe does not switch spaces, try these fixes:

- Make sure iss is open
- Check that trackpad gestures are enabled
- Confirm that macOS Spaces are turned on
- Close other apps that use swipe gestures
- Restart your Mac
- Open the app again after changing permissions

If the app opens but the swipe feels wrong:

- Check your trackpad speed and gesture settings
- Make sure you are swiping with three fingers
- Try a slower swipe
- Test both directions

If macOS blocks the app:

- Open Privacy & Security
- Allow the app to run
- Reopen iss after approval
